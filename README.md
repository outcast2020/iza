# iza
Iza é um chatbot de escrita reflexiva inspirado no modelo clássico de ELIZA, adaptado à cultura popular brasileira. Seu papel não é escrever pelo usuário, nem oferecer respostas prontas, mas perguntar a partir do que é dito, ajudando cada pessoa a pensar para escrever.
